Allgemeine Navigation

Die Navigation wird mittels der Funktionstasten durchgeführt. Im folgenden werden alle Funktionstasten aufgelistet, die zur Bedienung aller Fenster und Modie zuständig sind:

1. F1 Editor

Mit der Taste F1 wird der Editor geöffnet. Im Editor wird der Code geschrieben, der später ausgeführt werden soll.
----------------------------

2. F2 Console

Mit der Taste F2 wird die Console geöffnet. In der Console wird der geschriebene Code ausgeführt und wieder beendet.
----------------------------

3. F3 Hilfesystem

Mit der Taste F3 wird das Hilfesystem geöffnet. Dort kann man sämltiche Hilfetexte jeweils zu Lua und Basic je nach benutzem Modus finden und einige Grundlegenden Navigation und Bedienungsanleitungenfinden.
----------------------------

4. F4 Grafischeoberfläche

Mit der Taste F4 wird die Grafischeoberfläche geöffnet. Dort wird das jeweils programmierte Grafisch dargestellt.
----------------------------

----------------------------

6. F12 Herunterfahren

Mit der Taste F12 wird das System heruntergefahren.
